## About
It helps in saving your Codeforces profile in a text file.

## How to use?

- `git clone https://github.com/saurabhshadow/Automated-CF_Profile-Reports.git`
- `cd Automated-CF_Profile-Reports`
- Run `scrape.py`
- Now, enter your Codeforces user name.
